## 작성규칙  
- 폴더명은 영어로 한다.  
- 파일명은 (웬만하면) 한글로 한다.  

## 커밋 컨벤션
- `add:` → 새로운 파일이나 섹션 추가
- `update:` → 기존 파일의 수정, 설명 보완, 가독성 향상 등
- `fix:` → 버그, 오타, 잘못된 정보 수정
